## Commands

| command                        | function                                                 |
| ------------------------------ | -------------------------------------------------------- |
| vim -O [filename1] [filename2] | open file1 & file2 simultaneously and split horizontally |
| vim -o [filename1] [fliename2] | open file 1 & file 2 simultaneously and split vertically |



## Shortcuts

| key    | function |
| ------ | -------- |
| u      | undo     |
| ctrl+r | redo     |
|        |          |
